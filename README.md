# .NET 10 - Keyed Services Demo

A clean demonstration of **Keyed Services** (introduced in .NET 8) and modern Dependency Injection best practices in .NET 10.

## 📌 About This simple Project that explain the use of keyed service

This repository demonstrates how to effectively use **Keyed Services** to handle multiple implementations of the same interface — one of the most useful improvements in Dependency Injection in recent .NET versions.

### Key Features Showcased
- Keyed Services pattern
- Clean service registration using Extension Methods
- Strategy Pattern implementation
- Proper lifetime management
- Best practices for maintainable DI setup

## 🚀 Technologies Used

- **.NET 10**
- **ASP.NET Core Minimal APIs**
- **C# 14**
- Built-in Dependency Injection
